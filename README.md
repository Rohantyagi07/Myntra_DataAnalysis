# 🛍️ Myntra Fashion Dataset Analysis

> Exploratory Data Analysis (EDA) of the Myntra Fashion Dataset using Python to uncover trends in pricing, brands, product categories, ratings, and discounts.

---

## 📖 Overview

This project focuses on analyzing the Myntra Fashion Dataset using Python. The objective is to clean the dataset, perform exploratory data analysis (EDA), and visualize important insights that can help understand customer preferences, pricing strategies, and product distribution.

The project demonstrates practical data analysis skills including data preprocessing, visualization, and extracting business insights using popular Python libraries.

---

## 🎯 Objectives

- Clean and preprocess the dataset
- Handle missing and duplicate values
- Perform Exploratory Data Analysis (EDA)
- Analyze product pricing and discounts
- Explore brand and category performance
- Visualize trends using charts and graphs
- Generate actionable business insights

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/Myntra-Dataset-Analysis.git
```

### Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Run the Notebook

```bash
jupyter notebook myntra1.ipynb
```

Run all cells to reproduce the analysis.
